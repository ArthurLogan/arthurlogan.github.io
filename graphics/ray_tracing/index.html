    Ray Tracing

    光栅化无法解决全局的影响，因而提出光线追踪算法，基于三点假设。

    1. 光线沿直线传播。
    2. 光线之间不会相互碰撞。
    3. 光线从光源发出到达人眼，并且光路可逆。

    Whitted-Style RT

    从人眼发出“感知”光线和场景求交，计算在弹射过程中，每个交点上光线的颜色，递归叠加，得到像素的颜色。在这个过程中需要定义很多问题，我们依次介绍这些方法。

    • 光线求交，求交加速
    • 光线弹射，辐射度量学
    • 终止条件，只做镜面反射和折射
    • 路径追踪

    Uniform Spatial Partitions

    将场景切分成均匀的网格，每个网格记录表面和网格相交的物体。在”感知“光线发出后，逐个和网格求交，如果相交网格中包含物体，则与物体求交，否则得到下一个相交网格。

    Spatial Partitions

    使用不同的空间划分结构，将物体进行划分，如八叉树、KD树、BSP树等。

    对于KD树，每次选择一个轴进行切分，所有物体都只存储在叶子节点中，根据和网格相交决定是否向子节点搜索或和物体求交。但对于KD树的问题，很难判断三角形是否和网格相交，以及同一个物体可能和很多个网格相交。

    Bounding Volume Hierarchy

    将物体进行划分，每次划分后重新计算当前物体组的包围盒，每一个物体只会包含于一个包围盒内，根据包围盒的划分组成树状结构，并将物体存储在叶子节点，不同的叶子节点可能会产生相交。

    如何划分节点，一些启发点：

    1. 选择最长的轴进行划分。
    2. 取中间的物体进行划分，中位数，O(n)求第K大。
    3. 终止条件，比如叶子节点三角形数小于5。

    根据树状结构求交即可，和当前节点相交，则和所有子节点判断是否相交，再递归判断。

    Radiometry

    光照的描述和度量，从物理上定义了光照的属性：Radiant Energy，Radiant flux，intensity，irradiance，radiance。

    Radiometry

    Radiant Energy：电磁辐射的能量。

    Radiant Flux(Power)：电磁辐射通量、辐射功率，表示单位时间辐射的能量。

    Radiant Intensity：不同方向上的辐射通量，即单位时间、单位立体角的辐射能量。立体角是面积和半径平方的比值。 Irradiance：单位面积上接受到的辐射通量，即单位时间、单位面积上接受的能量。接受的辐射通量指的是和单位面积垂直的部分，类似Blinn-Phong里的Lambert's Cosine Law。

    Radiance：描述光线传播的属性，表示单位立体角、单位投影面积释放的辐射通量。表示的是面光源上朝着某个方向释放的通量强度。

    BRDF

    BRDF：前面的概念就是为了引出BRDF和后续的渲染方程。当物体的局部表面从某个方向上接受了Radiance，并向另一个方向上反射了Radiance时，BRDF描述这两者的关系。

    Path Tracing

    在Whitted-Style RT中，光线弹射会终止于漫反射表面，从而产生误差。路径追踪就是通过渲染方程求解正确的效果，其中包括求解积分和递归求解。

    Recursive Solving

    递归求解：在不考虑物体表面发光的情况，只考虑直接光照的部分，通过蒙特卡洛积分方法。 只考虑直接光照时，从相机发出的“感知”光线和场景求交，需要计算相交点朝向相机发出的 Radiance，即 BRDF 和局部接受的 Radiance 的积分，通过蒙特卡洛方法进行数值求解。如果采样方向没有光源，则无贡献，否则累加上述一项，其中：

    1. 光源的 Radiance
    2. 物体表面的 BRDF
    3. 入射光线和法向量
    4. 采样的 pdf

    均已知，则可以计算出上述采样值。进一步考虑间接光照，若 P 点采样得到的“感知”光线和场景相交于 Q 点，则 Q 点反射的光也会对 P 点朝向相机的 Radiance 产生贡献，该贡献值等价于相机从 P 点观察 Q 点，Q 点的直接光照朝向 P 点的 Radiance。

    2. 终止条件，俄罗斯轮盘赌，期望上正确。
    3. 效率太低，光源采样，将交点半球面的积分转换为在光源立体角上的积分。
